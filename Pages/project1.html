✦ Overview


For this project, my goal was to mock up the possibility of adding a webcam to the Friends of Schmeeckle website. This project focused more on the design aspect rather than an actual finalized project, so all of my deliverables highlighted that creative process to making a page for an already existing website. This project was done solo for a class where I learned about different design deliverables and the Web Design process.


✦ What I Overcame


A huge challenge for this project was creating a solution for a site with an existing design. Since my personal end goal was making a page that could be easily integrated into the original site, I wanted to focus on analyzing the preexisting design and creating a design that matched.


✦ Solution


My solution for this was focusing on deliverables such as moodboards to best represent the aesthetic I wanted for the design. That way, I could find a design that best represented Schmeeckle Reserve and the Friends of Schmeeckle website.
